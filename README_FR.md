# Module de Gestion des Patients README

## Aperçu

Ce projet consiste en la création d'un module de gestion des patients, conformément à la maquette fournie (voir `images/maquette.png`). Le module inclut les fonctionnalités de création, lecture, mise à jour et suppression des dossiers de patients (opérations CRUD).

## Technologies et Composants

Le projet doit être développé en utilisant les technologies et composants suivants :

- **PHP 7.4**
- **MySQL**
- **JavaScript**
- **Bootstrap**
- **Smarty Template Engine** : [Dépôt GitHub Smarty](https://github.com/smarty-php/smarty/tree/v5.3.1)
- **Template AdminLTE** : [Releases GitHub AdminLTE](https://github.com/ColorlibHQ/AdminLTE/releases)
